Class LoggerAppenderDailyFile

Description
Description | Vars (details) | Methods (details)

LoggerAppenderDailyFile appends log events to a file ne.

A formatted version of the date pattern is used as to create the file name using the http://www.php.net/sprintf function. <p>Parameters are $datePattern, $file. Note that file parameter should include a '%s' identifier and should always be set before $file param.</p>
