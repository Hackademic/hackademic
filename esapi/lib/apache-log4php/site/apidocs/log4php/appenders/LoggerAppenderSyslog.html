Class LoggerAppenderSyslog

Description
Description | Vars | Methods (details)

Log events using php http://www.php.net/syslog function.

Levels are mapped as follows:

  • level &gt;= FATAL to LOG_ALERT
  • FATAL &gt; level &gt;= ERROR to LOG_ERR
  • ERROR &gt; level &gt;= WARN to LOG_WARNING
  • WARN &gt; level &gt;= INFO to LOG_INFO
  • INFO &gt; level &gt;= DEBUG to LOG_DEBUG
